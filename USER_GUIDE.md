# SmartBrowser 智能浏览器 - 办公白领使用指南

欢迎使用 SmartBrowser！这份指南专门为您编写，即使您完全不懂代码，也能通过简单的步骤在您的电脑上运行并使用这款浏览器。

---

## 🚀 第一步：准备环境（只需操作一次）

就像安装 Office 软件一样，运行这款浏览器需要一个名为 "Java" 的运行环境。

1. **下载 Java 17**：
   - 请访问 [Oracle 官网下载页](https://www.oracle.com/java/technologies/downloads/#java17)。
   - 选择适合您电脑系统的版本（例如 Windows 用户的 `x64 Installer`）。
2. **安装**：
   - 下载完成后，双击安装程序，一路点击“下一步”直到安装完成。

---

## 📂 第二步：获取并准备浏览器

1. **获取软件**：
   - 您已经拥有了包含本指南的文件夹，这就是浏览器的所有文件。
2. **了解文件夹结构**（您不需要修改它们）：
   - `data/`：这里存放您的书签、历史记录等个人数据。
   - `src/`：这是浏览器的“零件库”。
   - `pom.xml`：这是浏览器的“说明书”。

---

## 🛠 第三步：如何运行（保姆级步骤）

因为软件还在开发阶段，我们通过一个简单的命令来启动它。

1. **打开命令行**：
   - **Windows 用户**：在文件夹的路径栏输入 `cmd` 然后回车；或者按下 `Win + R` 键，输入 `cmd` 回车。
   - **Mac 用户**：在文件夹上右键选择“在终端中打开”。
2. **输入运行命令**：
   - 在弹出的黑色窗口中，输入以下文字并按回车：
     ```bash
     mvn javafx:run
     ```
   - **注意**：第一次运行时，它会像下载更新一样下载一些必要的组件，请保持网络畅通。

---

## 🌟 第四步：功能介绍

启动后，您将看到一个现代化的浏览器界面：

1. **智能地址栏**：
   - 您可以直接在上方框内输入网址（如 `www.google.com`），也可以直接输入想搜索的内容（如“今天天气”），它会自动帮您跳转或搜索。
2. **书签管理**：
   - 看到喜欢的网页，点击地址栏右侧的“小星星”即可收藏。
3. **广告拦截**（即将上线）：
   - 它会自动帮您过滤网页上的烦人广告，让办公环境更清爽。
4. **历史记录**：
   - 忘记昨天看过的方案？按快捷键 `Ctrl + H` 即可查看。

---

## ❓ 常见问题

- **问：命令运行报错怎么办？**
  - 答：请检查您的 Java 是否安装成功，或者检查网络连接是否稳定。
- **问：默认搜索引擎是什么？**
  - 答：默认使用 Google 搜索，您也可以输入完整的网址直接访问。
- **问：我想保存书签，卸载重装后还在吗？**
  - 答：只要您保留了 `data/` 文件夹，您的个人数据就是安全的。
