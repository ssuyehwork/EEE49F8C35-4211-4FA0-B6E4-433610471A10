# SmartBrowser 用户使用指南 (Windows 修复版)

本指南旨在帮助您快速上手 SmartBrowser，并解决在 Windows 环境下可能遇到的乱码问题。

## 1. 快速启动 (推荐)

我们为您提供了一个一键启动脚本，它会自动配置环境并修复控制台乱码：

1.  在项目根目录下找到 `run_smartbrowser.bat` 文件。
2.  **双击运行** 即可。

---

## 2. 环境要求

在运行之前，请确保您的电脑已安装以下软件：

*   **JDK 17** 或更高版本 (建议使用 Zulu JDK 或 Oracle JDK)。
*   **Maven 3.6+** (用于管理项目依赖)。
*   **网络连接** (首次运行需要下载 JavaFX 库)。

---

## 3. 功能特性

*   **简洁界面**：顶层导航栏，支持后退、前进、刷新、停止和主页。
*   **多功能地址栏**：支持输入完整 URL 或关键词搜索 (默认使用 **Google**)。
*   **状态监控**：底部状态栏实时显示页面加载进度。
*   **编码优化**：针对中文环境进行了深度优化，自动注入字体修复网页乱码。
*   **去偏见化**：已移除所有对特定搜索引擎（如百度）的强制绑定，默认使用 Google。

---

## 4. 常见问题排查 (乱码专区)

如果您在运行过程中仍然看到乱码，请参考以下方案：

### Q1: 控制台日志显示 "???" 或奇怪的字符
*   **原因**：Windows CMD 默认代码页 (GBK) 与 Java 输出 (UTF-8) 不匹配。
*   **解决**：请务必使用 `run_smartbrowser.bat` 启动，或者在手动运行 `mvn` 前先输入命令 `chcp 65001`。

### Q2: 网页内容显示为方块或乱码
*   **原因**：JavaFX WebView 在某些 Windows 版本上无法正确识别系统字体。
*   **解决**：我们已在代码中增加了自动字体注入逻辑。如果问题依旧，请确保您的系统中安装了“微软雅黑” (Microsoft YaHei) 字体。

### Q3: 搜索时依然跳转到百度
*   **现状**：我们已在 v1.1 版本中彻底清理了所有百度引用。
*   **解决**：请运行 `mvn clean` 重新编译，确保使用的是最新代码。

---

## 5. 开发人员备注

如果您需要自行编译打包：

```bash
# 清理并打包
mvn clean package

# 运行 (带编码参数)
mvn javafx:run -Dfile.encoding=UTF-8
```
